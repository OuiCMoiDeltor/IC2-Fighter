\doxysubsection{Description détaillée}
Gestion des boutons dans l\textquotesingle{}interface utilisateur. 

Ce fichier contient toutes les fonctions nécessaires pour la création, la gestion et la destruction des boutons utilisés dans l\textquotesingle{}interface utilisateur du jeu. Il permet de définir les propriétés des boutons tels que leur texture, leur position, leurs états visuels (relâché, appuyé, survolé), ainsi que leur comportement interactif. Les fonctions supportent également la gestion dynamique de plusieurs états pour les boutons animés.
