Fichier gérant le menu principal du jeu \char`\"{}\+IC2 Fighter\char`\"{}. 

Ce fichier contient les déclarations nécessaires pour afficher et gérer le menu principal du jeu. Il inclut les interactions avec les boutons du menu pour naviguer dans les différentes options du jeu comme jouer, accéder aux options, ou quitter le jeu. Les ressources graphiques et sonores utilisées pour le menu sont également définies ici.
