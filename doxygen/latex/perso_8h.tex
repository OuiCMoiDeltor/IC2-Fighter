\doxysubsection{Description détaillée}
Fichier contenant les définitions de structures et les déclarations de fonctions pour la gestion des personnages dans le jeu. 

Ce fichier définit les structures nécessaires pour créer et gérer les personnages dans le jeu. Il contient également les déclarations des fonctions pour créer, mettre à jour et détruire les personnages, ainsi que pour gérer leurs interactions avec d\textquotesingle{}autres personnages et avec l\textquotesingle{}environnement.
