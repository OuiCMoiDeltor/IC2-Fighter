Description détaillée

Gestion du menu des options pour l'application SDL.

Ce fichier implémente les fonctions nécessaires pour gérer le menu des options, permettant aux utilisateurs de modifier la résolution de l'écran, le son, le mode plein écran et naviguer vers d'autres parties de l'application.

Gère les interactions dans le menu des options.

Cette fonction traite les événements utilisateurs, dessine les éléments de l'interface du menu des options et applique les modifications de configuration telles que la résolution de l'écran, le contrôle du son, et le mode plein écran.

Paramètres
eÉvénement SDL pour capter les actions de l'utilisateur.
rendererRenderer SDL pour dessiner les éléments du menu.
windowFenêtre SDL dans laquelle le menu est affiché.
largeurFPointeur vers la largeur actuelle de la fenêtre, peut être modifié pour changer la résolution.
hauteurFPointeur vers la hauteur actuelle de la fenêtre, peut être modifié pour changer la résolution.
scenePointeur vers la variable contrôlant l'affichage des différentes scènes de l'application.
sonPointeur vers la variable contrôlant l'état du son (activé/désactivé).
fullscreenPointeur vers la variable contrôlant le mode plein écran (activé/désactivé).
backgroundSoundPointeur vers la musique de fond pour gérer son arrêt ou sa lecture.
Renvoie
Un entier indiquant si l'utilisateur a quitté le menu des options (0 pour non, 1 pour oui).
